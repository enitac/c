First-Time Token Setup

Encrypt your GitHub Personal Access Token with a password. The encrypted token will be embedded in this HTML file.

This password lets you re-open Setup from the login page to change your token or password. Remember it.

How it works:
  1. Enter your GitHub Personal Access Token (starts with ghp_...)
  2. Choose a strong password — this becomes your login password
  3. Choose a Setup Protection Password — this lets you re-open Setup later to change your token or login password
  4. Click "Encrypt Token" — your token gets encrypted and automatically uploaded to your GitHub repo as a .token file
  5. You will be taken to the login page automatically
  6. From now on, just enter your password to log in — the encrypted token is fetched from GitHub each time

Note: Your raw token is NEVER stored anywhere. Only the AES-256-GCM encrypted version lives on GitHub. Without your password, it is useless.
